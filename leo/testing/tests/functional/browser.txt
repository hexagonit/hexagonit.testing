Post username and one file::

    >>> from StringIO import StringIO
    >>> data = {
    ...   'username': 'Some Name',
    ...   'my_text': {
    ...      'filename': 'file1.txt',
    ...      'content-type': 'text/plain',
    ...      'data': StringIO('Some Data')
    ...      }
    ... }

    >>> browser.post('/@@echo', data)
    >>> print browser.contents
    REQUEST_METHOD: POST
    ["my_text", "username"]
